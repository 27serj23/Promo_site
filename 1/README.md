# 🏠 Промо‑сайт — система управления промо‑кампаниями

Веб‑приложение на Django для организации поквартирных обходов, сбора статистики и ведения отчётности по промо‑кампаниям.

## 🚀 Возможности

- 🔐 Регистрация и авторизация пользователей (имя, фамилия, email, телефон)
- 📋 Создание промо‑кампаний и добавление адресов домов
- 👥 Добавление участников к кампании
- 🚪 Поквартирный обход:
  - отметка «открыли дверь / не открыли»
  - реакция (позитивно / нейтрально / негативно)
  - контакты (имя, телефон) и комментарий
- 📊 Статистика по кампании:
  - процент открытых дверей (в целом и по домам)
  - реакции (позитивные, нейтральные, негативные)
  - количество собранных контактов
- 🛠️ Административная панель Django для полного управления данными

## 🧱 Стек

- **Backend:** Python 3.12, Django 5
- **Frontend:** Bootstrap 5, HTML5, CSS3
- **База данных:** SQLite (легко сменить на PostgreSQL)
- **Безопасность:** переменные окружения через `python-decouple`
