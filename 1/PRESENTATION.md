 🏠 Промо‑сайт — система управления промо‑кампаниями

 🎯 Цель проекта

Создать веб‑приложение для организации и учёта промо‑кампаний, включающее регистрацию пользователей, управление адресами домов, фиксацию поквартирных обходов и вывод статистики.

 🛠️ Инструменты

- Python 3.12 + Django 5** — серверная часть
- Bootstrap 5 — адаптивный интерфейс
- SQLite — база данных
- Git — контроль версий

 📦 Основные сущности

1. Пользователи — фамилия, имя, email, телефон (обязательные поля)
2. Кампании — название, список домов, участники (ManyToMany)
3. Дома — город, улица, номер дома, подъезды, квартиры
4. Поквартирные обходы — факт открытия двери, реакция, контакты, комментарий

 🖥️ Демонстрация функционала

1. Регистрация и авторизация
2. Создание кампании
3. Добавление адресов домов
4. Приглашение участников
5. Фиксация обхода квартиры
6. Просмотр статистики (общая и по домам)
7. Админ‑панель для полного управления

 🧪 Тестирование

- Unit‑тесты покрывают создание кампаний, домов, обходов и расчёт статистики.
- Все тесты проходят успешно.

 💡 Особенности реализации

- Сервисный слой — бизнес‑логика вынесена из представлений
- Сворачиваемые формы — интерфейс не загромождён
- Безопасность — переменные окружения, защита от CSRF

 📈 Возможные улучшения

- Экспорт статистики в Excel
- Ролевая модель (администратор, менеджер, обходчик)
- Визуализация данных (графики)
- Отправка уведомлений участникам

 🏁 Итоги

Проект полностью соответствует учебному заданию, покрыт тестами и готов к демонстрации.
